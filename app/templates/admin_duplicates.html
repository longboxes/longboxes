Duplicates

Two kinds of duplicates show up here. Hash duplicates are the same archive (byte-identical) at multiple paths on disk — deletion is lossless once one path is kept. Issue duplicates are different files (different sha256) all claiming the same ComicVine issue, typically two scans or re-encodes of the same comic — we score each file by quality and highlight a recommended keeper. Actions are read-only plus an exclude from matching toggle; on-disk deletes are yours to do.

{# ==================== ISSUE DUPLICATES ============================== #}

Issue duplicates

Different files all matched to the same CV issue. Scoring order, best first: plausibility of being a real issue (15+ pages beats 5–14 beats <5) → ComicInfo coverage (full-with-CV-id > partial > none) → page resolution (interior sample preferred, cover as fallback) → file size → format (CBZ > CBR > CB7 > PDF) → raw page count → recency. The plausibility bucket exists so a 3-page sketch variant can't outrank a 25-page real comic on any combination of lower-tier signals.
