Archive backend

Which library reads CBZ / CBR / CBT / PDF files. comicbox (default) uses the third-party comicbox library — broader format support, proper page sort order, MetronInfo metadata extraction, and per-page metadata (double-page spreads, cover detection, skip-ads). stdlib falls back to the original zipfile / rarfile readers — narrower support but no extra dependencies. Switch if comicbox surprises you on a specific archive; the change is immediate (next scan / match / page view).

ComicVine cache

Marks every cached CV entity stale so the next page view triggers a fresh fetch from ComicVine. Data isn't deleted — file matches and your library stay intact — but covers, descriptions, and arc credits will refresh through the background worker as the rate limiter allows. Use this when CV has updated upstream and you want the app to pick up the changes without waiting for the TTL to expire.
